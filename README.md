# 📊 World Layoffs Data Cleaning & Exploratory Analysis

## 📌 Project Overview

This project focuses on cleaning, transforming, and analyzing global layoffs data using MySQL. The workflow includes data quality assessment, duplicate removal, standardization, null value handling, date transformation, and exploratory data analysis to uncover trends and patterns in layoffs across companies, industries, countries, and time periods.

The project demonstrates practical SQL techniques commonly used in real-world data cleaning and analytical workflows.

---

## 🎯 Project Objectives

- Prepare raw layoffs data for analysis through structured data cleaning techniques
- Identify and remove duplicate records
- Standardize inconsistent values and formats
- Handle missing and blank values appropriately
- Perform exploratory analysis to identify layoff trends and business patterns
- Generate analytical insights using SQL window functions and Common Table Expressions (CTEs)
